A game that design shoes and can make changes on them and check their information.

[USAGE]

CREATE SHOES:
shoes {type of shoe} {type of shoe} {type of shoe}
sport shoe {size} {ID} {body color} {shoelase color} {soles color} : creates sport shoes
female shoe {size} {ID} {body color} {heel color} {heel length} : creates female shoes
male shoe {size} {ID} {body color} {soles color} : creates male shoes

EDIT SHOES INFORMATION:

Edit : Edit information

SHOW SHOES INFORMATION:

Shoe : Show a special shoe information

History : Show all shoes information
